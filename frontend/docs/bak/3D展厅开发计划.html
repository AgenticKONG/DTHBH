1. 项目概览

1.1 项目目标

  • 创建一个高质量的3D数字展厅，展示黄宾虹的艺术作品
  • 实现沉浸式的浏览体验，参考国博数字展厅的交互方式和空间设计
  • 支持多展厅切换（序厅、早期展厅、盛期展厅、晚期展厅、尾厅）
  • 提供流畅的用户交互和视觉效果，真实复现线下展览体验
  • 每个展厅包含多个展示区域，如国博展厅的多层次结构

1.1.1 国博数字展厅参考要点

国博展厅结构分析：
  • 序厅 - 入口大厅，提供展览导览和背景介绍
  • 主题分区 - 每个部分包含多个子主题区域
  • 互动展示 - 使用互动屏等多媒体形式
  • 空间层次 - 通过隔断、拱门、柱子划分空间
  • 命名风格 - 富有文学性和艺术性的展厅名称

1.3 观展路线设计

观展路线原则：
  • 顺序性：从序厅→早期→盛期→晚期→尾厅，每个展厅内部按时间顺序参观
  • 引导性：使用隔断墙、拱门、柱子等元素引导观展路线
  • 探索性：S形或Z形路线，避免直线型布局的单调
  • 完整性：确保观众不会错过任何展区

2. 需求分析

2.1 功能需求

  1. 观展路线设计
    • 整体路线：序厅 → 早期展厅 → 盛期展厅 → 晚期展厅 → 尾厅
    • 展厅内路线：入口 → 区域1 → 区域2 → 区域3 → 出口（按时间顺序）
    • 路线引导：使用隔断墙、拱门、柱子等元素引导观展路线
    • S形/Z形路线：避免直线型布局，增强探索性和引导性
  2. 场景展示
    • 序厅：入口大厅，提供展览导览和黄宾虹简介（宽24米，长50米，高10米）
    • 早期展厅 - 疏淡清逸：展示白宾虹时期作品（1865-1930）
      • 区域1：宗法新安（1870-1890）- 介绍早期学习
      • 区域2：花鸟探索（1890-1910）- 展示花鸟画作品
      • 区域3：画风转变（1910-1930）- 展示由白入黄的过渡
    • 盛期展厅 - 浑厚华滋：展示黄宾虹时期作品（1930-1948）
      • 区域1：五笔七墨（1930-1940）- 介绍理论创新
      • 区域2：黄山情结（1935-1948）- 展示黄山主题作品
      • 区域3：浑厚华滋（1940-1948）- 展示成熟期作品
    • 晚期展厅 - 黑密厚重：展示黑宾虹时期作品（1948-1955）
      • 区域1：积墨大成（1948-1952）- 展示积墨技法作品
      • 区域2：焦墨宿墨（1952-1954）- 展示焦墨宿墨作品
      • 区域3：艺术升华（1954-1955）- 展示晚期巅峰作品
    • 尾厅：艺术总结，展示黄宾虹的艺术成就和影响
    • 建筑细节：地面、墙壁、柱子、拱门、踢脚线、天花板、隔断、展示墙
    • 展品展示：画框、画作、说明牌、互动展示屏、介绍面板
  3. 交互控制
    • 键盘移动：W/A/S/D 或 方向键
    • 鼠标控制：拖动旋转、滚轮缩放
    • 点击交互：点击入口标记切换展厅，点击画作查看信息，点击说明牌了解详情
    • 导航模式：自主漫游、展览构成、三位场景三种模式切换
    • 自动导览：按指定路线自动浏览（可选）
  4. 视觉效果
    • 真实的光影效果（阴影、反射、高光）
    • 合适的材质（金属、木材、纸张、石材）
    • 氛围效果（尘埃粒子、光线束）
    • 空间层次感（隔断、拱门、柱子划分区域）

2.2 非功能需求

  • 性能：帧率 ≥ 30fps，加载时间 < 5秒
  • 兼容性：支持主流浏览器（Chrome、Firefox、Safari）
  • 响应式：适配不同屏幕尺寸
  • 用户体验：操作流畅，无明显卡顿

3. 技术设计

3.1 技术栈

  • 3D引擎：Three.js (r128+)
  • 控制器：OrbitControls
  • 渲染器：WebGLRenderer
  • 材质：MeshStandardMaterial（支持光照）
  • 阴影：PCFSoftShadowMap
  • 框架：Vue 3

3.2 架构设计

ThreeDExhibition.vue ├── 初始化 (init) │ ├── 创建场景 │ ├── 创建相机 │ ├── 创建渲染器 │ ├── 创建控制器 │ └── 加载配置 ├── 场景构建 │ ├── createHall() - 创建展厅 │ ├── addLights() - 添加灯光 │ ├── createArtworkMarkers() - 创建展品 │ ├── createEntranceMarkers() - 创建入口标记 │ └── createAtmosphereEffects() - 创建氛围效果 ├── 交互控制 │ ├── 键盘事件 │ ├── 鼠标事件 │ └── 射线检测 ├── 动画循环 │ ├── 控制器更新 │ ├── 相机移动 │ ├── 动画效果 │ └── 渲染场景 └── 资源管理 ├── 切换展厅 └── 清理资源

3.4 灯光系统

  • 环境光：0xfff5e6, intensity 0.5（柔和基础光）
  • 主方向光：0xfff8e1, intensity 1.0（主光源，带阴影）
  • 补光：0xffffff, intensity 0.4（填充阴影）
  • 聚光灯：0xfff5e0, intensity 2.0（照亮画作，带阴影）
  • 壁灯：0xfffaf0, intensity 0.8（沿墙壁）
  • 底部反射光：HemisphereLight（增加地面反射感）

  • ☐ 材质质感是否真实（木纹、金属光泽等）
