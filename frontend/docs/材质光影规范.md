## 一、材质设计原则

### 1.1 整体风格
- **风格定位**: 博物馆氛围，庄重典雅
- **色彩基调**: 米白色系，暖色调为主
- **质感特点**: 细腻、温润、自然
- **统一性**: 各展厅保持风格统一，但各有特色

### 1.2 设计原则
- **真实性**: 材质符合物理规律
- **艺术性**: 突出展品，不喧宾夺主
- **耐久性**: 考虑长期展示的视觉效果
- **可维护性**: 便于清洁和更新

## 七、光影设计原则

### 7.1 整体光影风格
- **风格定位**: 温暖、柔和、有层次
- **主光源**: 暖白色，营造温馨氛围
- **辅助光**: 冷白色，增加层次感
- **阴影**: 柔和自然，不生硬

### 7.2 设计原则
- **真实性**: 光影符合物理规律
- **艺术性**: 突出展品，营造氛围
- **节能性**: 合理使用光照资源
- **可调性**: 支持光照强度调节

## 十一、光影性能优化

### 11.1 性能优化策略

1. **距离剔除**：远距离物体不接收光照
2. **视锥剔除**：视锥外的物体不计算光照
3. **LOD光照**：根据距离调整光照质量
4. **烘焙光照**：静态物体使用烘焙光照
5. **延迟渲染**：大量光源时使用延迟渲染

### 11.2 光照质量设置

| 质量 | 环境光 | 聚光灯 | 阴影 | 体积光 | 适用设备 |
|------|--------|--------|------|--------|---------|
| 低 | 0.3 | 1.0 | 512×512 | 关闭 | 移动设备 |
| 中 | 0.4 | 1.5 | 1024×1024 | 简化 | 中端设备 |
| 高 | 0.5 | 2.0 | 2048×2048 | 标准 | 桌面设备 |
| 超高 | 0.6 | 2.5 | 4096×4096 | 高级 | 高端设备 |

### 11.3 动态调整

```javascript
// 根据设备性能动态调整光照
function adjustLightingQuality(performance) {
  if (performance === 'low') {
    // 降低光照质量
    ambientLight.intensity = 0.3;
    spotLight.intensity = 1.0;
    shadowConfig.mapSize = { width: 512, height: 512 };
    volumetricLight.enabled = false;
  } else if (performance === 'high') {
    // 提高光照质量
    ambientLight.intensity = 0.5;
    spotLight.intensity = 2.0;
    shadowConfig.mapSize = { width: 2048, height: 2048 };
    volumetricLight.enabled = true;
  }
}
```
