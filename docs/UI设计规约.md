# 數字黃賓虹 (Digital Huang Binhong) UI 設計規約

本規約旨在規範 Agents 在開發過程中的視覺產出，確保「金石美學」與「水墨質感」在所有模塊中高度統一。

---

## 1. 核心配色方案 (Color Palette)

### 1.1 紙感與基調 (Backgrounds)
- **主背景色**: `#fdf5e6` (OldLace) - 模擬陳年宣紙色。
- **次級背景**: `#f4efdf` - 用於導航欄或次要區域。
- **宣紙紋理**: `url('https://www.transparenttextures.com/patterns/parchment.png')` - 全局覆蓋。

### 1.2 墨感與文字 (Typography & Ink)
- **主文字色**: `#3d2b1f` (深褐墨色) - 用於正文和標題。
- **次要標籤**: `#8b7d6b` - 用於日期或輔助註釋。
- **淡墨輔助線**: `rgba(61, 43, 31, 0.15)` - 用於橫軸、邊框或分隔線。

### 1.3 金石紅 (Accent Color)
- **主活動色**: `#c0392b` (絳紅) - 用於印章、選中狀態、關鍵路徑。
- **高亮陰影**: `rgba(192, 57, 43, 0.2)` - 用於選中卡片的發光感。

---

## 2. 字體與排版 (Typography & Layout)

### 2.1 字體族
- **標題與抒情**: `"Ma Shan Zheng", "KaiTi", "STKaiti", cursive, serif`。
- **正文與史料**: `"SimSun", "STSong", "STKaiti", serif` - 強調學術穩重感。
- **年份與數字**: `"Georgia", "Times New Roman", serif`。

### 2.2 佈局範式
- **「左圖右史」模式**: 左側佔據 70%~78% 用於長卷/地圖，右側 22%~30% 用於垂直流式詳情面板。
- **空間呼吸**: 在長卷的起點 (1864) 和終點 (1955) 必須預留至少 60px 的物理留白。
- **垂直鎖定**: 全局地圖視窗應實現垂直鎖定，嚴禁雙軸滾動。

---

## 3. 組件設計語言 (Component Design)

### 3.1 卡片與面板
- **印章標籤**: 位置、地點應採用 `writing-mode: vertical-rl` 的豎排格式，背景色為絳紅 (#c0392b)。
- **宣紙卡片**: 白色背景 (#fff) 配合極細邊框 (#d2b48c)，並帶有 `2px 2px 5px rgba(0,0,0,0.03)` 的微弱投影。

### 3.2 交互與導航
- **懸浮工具條**: 採用 `backdrop-filter: blur(8px)` 的毛玻璃效果，背景色為 `rgba(255, 255, 255, 0.7)`。
- **水墨滾軸**: 寬度 8px-12px，顏色統一為 `#d2b48c` (赭石)，滑塊應有明確的圓角。

---

## 4. Agents 執行守則 (Implementation Rules)
1. **嚴禁純白**: 除非是卡片底色，否則背景嚴禁使用 `#fff` 或 `white`。
2. **座標對齊**: 時代環境大事必須以「垂直指示條」形式精確對齊年份 X 軸。
3. **溢出保護**: 所有 D3 繪圖必須具備至少 1.5x 的數據緩衝，波峰嚴禁觸頂。
