# 墨海星圖：數字社交網絡設計思路

## 1. 核心定位 (Core Identity)
墨海星圖（Ink Sea Map）是黃賓虹社交網絡的 **“宏觀視角”**。通過力導向圖（Force-Directed Graph）展現賓翁交遊的廣度與時間維度上的動態生長。

## 2. 視覺語言 (Visual Aesthetics)
*   **印章地理**：將北平、上海、杭州、四川等核心地理座標抽象為 **“紅色朱砂印章”**，作為引力場中心。
*   **星途路徑**：人物節點像繁星般環繞在地理中心周圍，通過連線的粗細與虛實表現互動的頻率與性質。
*   **水墨擴散**：節點與路徑應用 `#ink-spread-ya` 濾鏡，模擬宣紙滲墨的自然質感。

## 3. 交互邏輯 (Interaction Logic)
*   **時間滑塊驅動**：用戶撥動生平紀元滑塊，節點會隨年份“生長、遷移、消散”。
*   **地理跳變**：當賓翁發生重大移居（如 1937 入蜀）時，賓翁節點會發生劇烈的地理位移，帶動周邊社交網絡的重構。
*   **敘事卡片對齊**：點擊人物節點，彈出 **“右側扉頁式敘事卡片”**。其結構、色調、豎排書法風格與“知音對話”長卷完全一致，確保全站用戶體驗的連貫性。

## 4. 數據支撐 (Data Foundation)
*   **動態頻率**：節點大小由該年份的互動頻率（Frequency）決定。
*   **物理隔離**：墨海星圖侧重於“廣度”，與侧重“深度”的知音長卷通過相同的 `hbh_location_history.json` 公理軌跡進行時空同步。
