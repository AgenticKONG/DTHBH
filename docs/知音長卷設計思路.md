# 黃賓虹數字社交網絡：知音長卷設計思路

## 1. 核心哲學 (Artistic Identity)
本組件旨在通過“長卷”形式，還原黃賓虹與核心知音（如傅雷、許承堯）之間跨越數十年的精神共振。
*   **信（史實真實）**：以黃賓虹真實地理位移為物理基線，區合同城與異地。
*   **達（情感真實）**：通過視覺隱喻（磁吸、糾纏）表現文字背後的靈魂引力。

## 2. 視覺隱喻系統 (Visual Metaphors)
*   **物理基線**：焦墨（賓翁）與石青（知音）兩線在異地時平行分立，象徵空間阻隔。
*   **相契晤談 (同城)**：兩線在“聚”的瞬間產生 **DNA 狀螺旋糾纏**，象徵生命能量的深度交織。
*   **錦書遙寄 (書信)**：異地通信時，兩線向中軸線 **“磁吸下探”**，產生瞬間的聚攏效果，寓意“見字如面”。
*   **宿命初識**：第一筆交互點使用 **超大墨滴 (Ink Drop)** 效果，象徵敘事的開啟。
*   **物理終點**：1955年3月25日節點使用 **虛線殘墨印章**，標記為“終”，寓意哲人萎謝。

## 3. 交互與敘事邏輯 (UX & Narrative)
*   **數智化標識**：節點內統一顯示 **交互密度數字**。數字越大，印章尺寸越大、墨暈越濃。
*   **文學化標籤**：側邊欄廢除“兩人”、“互動”等生硬詞彙，改用“對影晤談”、“莫逆相契”、“尺牘頻傳”等詩意表達。
*   **二級研讀模式**：
    *   第一級：快速卡片，顯示金句與摘要。
    *   第二級：全屏 **“宣紙長札”**，提供完整史料原文與文獻出處，服務於專業研究者。

## 4. 特殊設計：傅雷的回響 (Echoes of Fu Lei)
*   **跨時空回溯**：針對傅雷特有的 1955 年後評價史料，設計 **“回響虛節點”**。
*   **溯源弧線**：點擊後世評價，產生一條指向往年對應事件的淡色弧線，表現“人琴俱亡，知音回響”的時空跨度。

## 5. 數據工程 (Data Engineering)
*   **解耦設計**：將黃賓虹地理軌跡提煉為獨立的“公理 JSON”，供所有知音視圖共享，確保背景基線絕對統一。
*   **動態時期**：背景色塊（上海、北平、杭州）由地理公理 JSON 動態驅動，與長線轉折精準對齊。
