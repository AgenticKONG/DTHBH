# 《数字黄宾虹 · 1932 蜀游长卷》设计思路

## 1. 核心设计哲学 (Artistic Identity)
本专题旨在以数字手段还原黄宾虹先生 1932-1933 年入蜀游学的时空轨迹，通过“数字化长卷”的交互形式，让用户从地理、艺术、人文三个维度进行沉浸式赏析。

### 1.1 卷轴叙事 (Scroll Narrative)
*   **交互隐喻**：采用水平滚动逻辑（从右至左），模拟中国传统手卷的展开方式。
*   **三位一体**：将“地理空间路径”（上）、“人文题跋卡片”（中）、“时间线性刻度”（下）高度联动。

### 1.2 笔随心动 (Dynamic Expression)
*   **笔墨提按**：通过实时计算用户的滚动速度，动态调整线条宽度（1.5px - 6px），模拟毛笔运笔中的提按变化。
*   **飞白意境**：当用户快速疾行（高速滚动）时，通过透明度随机扰动与滤镜毛糙度增强，模拟水墨画中的“飞白/枯笔”质感。

---

## 2. 视觉美学实现 (Aesthetics)
*   **水墨滤镜**：基于 SVG `feTurbulence` 和 `feDisplacementMap` 构建 `ink-spread-ya` 滤镜，模拟墨汁在宣纸纤维上的自然晕散。
*   **宣纸背景**：全局注入纤维纹理（Rice Paper Texture），提升页面的历史厚重感。
*   **姓名印章**：设计 2x2 方正布局的“黃賓虹印”，通过 Grid 布局确保传统朱文印章的布局严谨性。

---

## 3. 关键技术方案 (Technical Implementation)

### 3.1 江流曲折算法 (Meandering Path)
*   **Catmull-Rom 插值**：使用 D3.js 的 `curveCatmullRom` 曲线算法。
*   **自然波动**：在固定站点之间动态注入 3-4 个基于正弦波的中继波动点，模拟长江九曲回肠的地理形态，打破纯数学曲线的单调感。

### 3.2 进退判定逻辑 (Directional Logic)
*   **进笔（前进）**：激活艺术效果（提按、飞白），增强叙事感。
*   **收笔（后退）**：回归标准稳定线条，确保撤销/回溯动作的视觉整洁。

### 3.3 沉浸式卷轴鉴赏 (Appreciation Mode)
*   **展卷动画**：利用 Vue `<transition>` 实现画作从中心呈卷轴状缓缓展开。
*   **托裱质感**：模拟“绫边托裱”画框，并采用竖排书法题跋，提供博物馆级的读画体验。
